# Assignments

Course assignments for Algorithms Design & Analysis, Machine Intelligence and Computer Architecture.

This repository contains selected course assignments completed during my undergraduate studies in Computer and Communications Engineering.  
The assignments focus on practical problem-solving, algorithmic thinking, and applying theoretical concepts through implementation.

## Contents

- Trees and Graphs
- Hashing and Sorting
- Complexity analysis and optimization

Through these assignments, I developed the ability to:
- Break down complex problems into smaller subproblems, becoming proficient in solving common to medium-level brute-force and recursive problems
- Choose appropriate data structures to improve efficiency
- Analyze time and space complexity
- Translate theoretical algorithms into working code, such as Quick Sort, Heap Sort, Selection Sort, Insertion Sort, Bubble Sort, and Dijkstraâ€™s algorithm

### Machine Intelligence
This section contains assignments related to introductory machine intelligence topics, including:
- Search algorithms
- Heuristics
- Reinforcement learning concepts
- Problem modeling and evaluation
- Basic decision-making algorithms

Through these assignments, I gained experience in:
- Modeling problems mathematically and computationally
- Implementing learning-based and search-based solutions
- Understanding the trade-offs between different intelligent approaches

### Computer Architecture
This section includes assignments and lab work focused on **digital design and processor fundamentals**, implemented using **VHDL** and simulated using **QuestaSim / ModelSim**.

Topics and projects covered include:
- Designing Finite State Machines (FSMs) for control logic
- Implementing sender and receiver FSMs for digital communication
- Designing and testing a basic 16-bit ALU
- Developing a generic ALU supporting arithmetic, logic, and shift operations
- Writing testbenches to verify correctness and functionality
- Simulating and debugging designs using waveform analysis in QuestaSim

Through these assignments, I developed skills in:
- Hardware description using VHDL
- Modular digital design and control logic
- Writing and validating testbenches
- Understanding the interaction between datapath and control units
- Translating architectural concepts into synthesizable hardware designs
